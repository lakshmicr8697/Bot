**Discord AI Image Captioning Bot**

**Features**
	• Accepts image uploads in Discord
	• Generates captions and tags using Hugging Face model
	• Replies automatically in the Discord channel
	
**Tech stack**
	 Python + discord.py + Transformers + ViT-GPT2 image captioning
	 
**How to run**
	pip install -r requirements.txt
	create .env and add:
	DISCORD_TOKEN=YOUR_TOKEN
	python bot.py
